Beef, Poultry, and Pork Production – May 2015

This map the user the option to compare the amount of either beef, pork, or poultry by each state and the ammount of beef, pork, and poultry within each state. The data is presented in the form of a classed choropleth divided up by the precentages of total agricultural revenue per state.

This map will allow the user to communicate with an user interface to switch between beef, pork, and poultry.

Energy Production in the US – 2015

This map allows users to see data on various power plants by fuel source type including some production information.

This map functions by taking arrays of data and drawing a circly around features of that data that vary by size of production of the power plant. It also allows the user to click on locations.
